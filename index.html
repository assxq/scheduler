SYNC / DATA

최초 1회만: Supabase 대시보드 → SQL Editor에 아래를 붙여넣고 실행하세요. 
create table if not exists scheduler_state (
  id int primary key,
  data jsonb,
  updated_at timestamptz default now()
);
alter table scheduler_state enable row level security;
create policy "sched anon all" on scheduler_state
  for all using (true) with check (true);
alter publication supabase_realtime add table scheduler_state;
⚠ anon key로 누구나 읽고 쓸 수 있는 개인용 설정이에요. URL·키를 외부에 공유하지 마세요.
